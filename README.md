# Классификация клеток крови (острый лимфобластный лейкоз)

Проект по классификации медицинских изображений: определение типа клеток крови при остром лимфобластном лейкозе (ОЛЛ). Сравниваются три архитектуры: самописная CNN, CNN с Dropout и предобученная ResNet18.

**Ссылка на проект:** [blood_cell_cancer_classification.ipynb](https://colab.research.google.com/drive/1BK6i3_er3pRJCgAogXDGduMRY9zeWUtv#scrollTo=U2H5NBzNoCqz)
**Источник данных:** [Blood Cells Cancer (ALL) dataset](https://www.kaggle.com/datasets/mohammadamireshraghi/blood-cell-cancer-all-4class)

---

## Этапы работы

**Технологии:** Python, PyTorch, Torchvision, Scikit-learn, NumPy, Matplotlib, Seaborn

### 1. **Загрузка и подготовка данных**
   - Распаковка датасета с изображениями клеток крови (4 класса)
   - Разделение на train/test в пропорции 80/20
   - Контроль распределения классов

### 2. **Аугментация и предобработка**
   - Изменение размера до 128×128 пикселей
   - Нормализация значений пикселей в диапазон [-1, 1]
   - Аугментация: случайные повороты (±10°) для увеличения вариативности
   - Разделение train на train/val (80/20)

### 3. **Моделирование**
   Сравнила 3 архитектуры:
   - **Simple CNN:** 3 сверточных слоя с BatchNorm
   - **CNN + Dropout:** аналогичная архитектура с Dropout (0.2-0.3)
   - **ResNet18:** предобученная на ImageNet модель (fine-tuning)

### 4. **Обучение**
   - 10 эпох для каждой модели
   - Сохранение лучших весов (по F1-score на валидации)
   - Визуализация динамики loss и F1-score

### 5. **Результаты**
   - **Simple CNN:** F1 = 0.994
   - **CNN + Dropout:** F1 = 0.977
   - **ResNet18:** F1 = **0.998**

---

## Сравнение моделей

| Модель | Accuracy | F1-score |
|--------|----------|----------|
| Simple CNN | 0.99 | 0.994 |
| CNN + Dropout | 0.98 | 0.977 |
| ResNet18 | **1.00** | **0.998** |


---

## Процесс обучения

### Simple CNN
- Стабильное обучение, но скачки loss на валидации
- Лучший F1: 0.994

### CNN + Dropout
- Медленнее учится из-за регуляризации
- Нестабильность на эпохе 9 (loss=6.08, F1=0.481)
- Лучший F1: 0.977

### ResNet18
- Быстрая сходимость (уже с 1-й эпохи F1 > 0.9)
- Стабильные результаты
- Лучший F1: 0.998

---

## Что я узнала

1. **Transfer Learning эффективен:** ResNet18 показала лучший результат (F1=0.998) благодаря предобученным весам
2. **Dropout не всегда помогает:** на этом датасете Dropout ухудшил результаты (F1=0.977 против 0.994)
3. **Класс benign сложнее всех:** все модели хуже классифицировали доброкачественные клетки (их меньше в датасете)
4. **Медицинские изображения хорошо классифицируются** при правильной предобработке

**Ключевой вывод:** Предобученные модели значительно эффективнее самописных, достигая почти идеальной точности на данном датасете.
